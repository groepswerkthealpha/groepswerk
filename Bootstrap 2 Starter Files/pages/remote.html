Communication

The longboard an remote communicate through
bluetooth. This is what makes you able to use
the provided app. Just don't try to make the app by yourself,
because there is a security code that is unique to each board.

Display

The remote is equipped with a 1.3" OLED display.
The display is able to show quite a few things, such as:
speed, battery levels, temperatures, if longboard is connected or not,
the current mode,...
